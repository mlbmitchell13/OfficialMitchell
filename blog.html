My First Software Internship!

My first software internship is one that I did not expect to go the way that it did. Initially, I was hired as an automation engineering intern who was expected to only learn about SaltStack and Splunk. To my surprise, eventually Python and HTML/CSS came into play as well.

During the first month, I was introduced to servers and server management. The major server management system First Data uses is SaltStack. SaltStack, in simple terms, is having, what is called a "master" server, that talks and sends commands to other servers called "minions.” Minions, which are also called agents, are connected to the master by exchanging private keys and listening on ports 4505/4506. They are each installed on multiple machines. An example of how the master communicates to its minions is when you want to update/install thousands of servers at once rather than one-by-one. The master would use send Salt command to a certain minion(s) and have them install/update themselves depending on their operating system. This also leads me to my next point. SaltStack has the ability to manage servers of all operating systems. Whether it be Windows or Linux/Linux Flavors, SaltStack makes it relatively easy to manage thousands of servers at once, hence automation engineering.

Entering the second month, the interns and I were given a project by the CTO Mike Manos. This project entailed developing code that used the Twitter API (Tweepy) to grab tweets, run them through sentiment analysis, and find negative tweets about the company. This information is useful because it allows the company to see issues clients are having outside of their customer service line. A lot of issues that clients have are presented through social media and by developing code to catch those negative tweets, it allowed the company's L1 and L2 teams to respond faster and increase overall customer satisfaction.

How does HTML/CSS and Python come into play? The CTO’s project was created using Tweepy which requires the usage of Python. With Python, we created a few scripts to pull tweets from First Data clients such as Walmart, Keller Williams, etc. HTML/CSS was used to develop the User Interface for the software. Mike wanted the interns to create a dashboard of some sort to display the tweet data pulled in. We decided to us HTML/CSS so it would be accessible from the internet and be displayed as a webpage.

Overall, my internship with First Data was an incredible opportunity to not only learn some new skills, but also build upon the ones I already have. Very excited to see where I end up next!
